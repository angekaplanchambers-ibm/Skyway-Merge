## Scope

This artifact turns the three architecture variants into Mermaid diagrams:

1. Current Terraform today, with Infragraph shown as an external surface.
2. Option A - org-first.
3. Option B - workspace-centric.

## Interpretation Notes

- The baseline uses the current Terraform navigation from the local UI reference.
- Infragraph stays outside the core Terraform tree because the workshop baseline separates it from Terraform's current navigation.
- Option B follows the corrected reading in `001.26.04.02.Skyway-Merge-IA-Diagrams.md`: no new multi-org dashboard, workspace is the main operating surface.
- The older prototype plan includes an Option B org dashboard route for testing. This diagram set follows the corrected IA note instead of that older route scaffold.

## Diagram 1 - Current Terraform Today + External Infragraph

```mermaid
flowchart TB
    entry["Organizations"]

    subgraph terraform["Terraform Today"]
        org["Organization"]
        workspaces["Workspaces"]
        workspace["Workspace"]
        overview["Overview"]
        runs["Runs"]
        states["States"]
        searchimport["Search & Import"]
        projects["Projects"]
        stacks["Stacks"]
        registry["Registry"]
        explorer["Explorer"]
        usage["Usage"]
        settings["Settings"]

        org --> workspaces
        workspaces --> workspace
        workspace --> overview
        workspace --> runs
        workspace --> states
        workspace --> searchimport
        org --> projects
        org --> stacks
        org --> registry
        org --> explorer
        org --> usage
        org --> settings
    end

    subgraph external["Separate HCP Surface"]
        dashboard["HCP Dashboard"]
        project["Project"]
        infragraph["Infragraph"]
        iam["IAM"]
        hcpsettings["HCP Settings"]

        dashboard --> project
        project --> infragraph
        project --> iam
        project --> hcpsettings
    end

    entry --> org
    org -. separate path .-> dashboard

    classDef core fill:#EAF2FF,stroke:#355C7D,color:#0F172A,stroke-width:1.5px;
    classDef external fill:#FFF3D6,stroke:#B45309,color:#451A03,stroke-width:1.5px;
    class entry,org,workspaces,workspace,overview,runs,states,searchimport,projects,stacks,registry,explorer,usage,settings core;
    class dashboard,project,infragraph,iam,hcpsettings external;
```
